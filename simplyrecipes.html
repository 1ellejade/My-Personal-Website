Simply Recipes Web Scraping Project

My very first solo endeavor with Python!

Jun-Aug 2021 Gabrielle Ebbrecht

Have you ever looked for recipe instructions online? If so, I'm sure you've wasted both time and energy looking through food bloggers' childhood stories of the meal you're looking to make... sometimes a lengthy history of its origins and culture. I know I don't want to have to scroll through their whole post looking for a list of ingredients, or the steps to make my meal, so I decided to program a web scraper that will get rid of all the fluff for me.

I utilized beautifulsoup4 to scrape the HTML code from the SimplyRecipes website. If the program caught any of my inputted allergies in my recipe of choice, it would raise a warning and ask permission to save the recipe before appending the ingredients and instructions to a recipe file for ease of reference. While the output wasn't anything crazy, the satisfaction of getting the code to run was enough to solidify my interest in programming.
